<h2>🧠 Concepto: Estructuras Secuenciales</h2>
<p>Las <strong>estructuras secuenciales</strong> son aquellas en las que las instrucciones se ejecutan una tras otra, siguiendo un orden lógico y lineal, sin condiciones ni repeticiones. Son la base de cualquier programa, ya que permiten realizar pasos definidos de forma ordenada.</p>
</div>

<div class="card">
<h2>📋 Ejemplos</h2>
<ol>
<li><strong>Ejemplo 1:</strong> Calcular el área de un rectángulo.
<pre><code>Leer base y altura
Calcular área = base * altura
Mostrar área</code></pre></li>

<li><strong>Ejemplo 2:</strong> Mostrar un mensaje de bienvenida.
<pre><code>Imprimir "Bienvenido al sistema"</code></pre></li>

<li><strong>Ejemplo 3:</strong> Sumar dos números.
<pre><code>Leer A, B
Calcular S = A + B
Mostrar S</code></pre></li>
</ol>
